# Этап: Editor Commands

На этом этапе в редактор Web-SCADA Lite добавлен базовый профессиональный набор операций:

- множественное выделение (`Ctrl/Shift + Click`, рамка выделения);
- группировка и разгруппировка;
- блокировка и разблокировка объектов (`locked`);
- выравнивание по сторонам и центрам;
- приведение к одинаковому размеру;
- распределение и равномерные интервалы;
- горячие клавиши для основных команд.

Архитектурно операции вынесены в слой чистых функций (`packages/shared/src/editor-commands.ts`),
что готовит основу для следующих этапов: `undo/redo`, `copy/paste`, расширенные хоткеи и контекстные команды.
